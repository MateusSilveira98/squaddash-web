# squaddash-web

## Stacks
```
node: v10.6.0
vue
json-server (mock data)
```

## Project setup
```
npm install
npm i -g json-server
```

### Compiles and hot-reloads for development
```
npm start
```
### Run mock datas for development
```
json-server db.json
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Run your unit tests
```
npm run test:unit
```
